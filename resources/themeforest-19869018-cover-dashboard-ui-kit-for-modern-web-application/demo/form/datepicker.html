Flatpickrhttps://github.com/chmln/flatpickr

DatePicker is a pre-built component bases on Flatpickr

<date-picker value="2017-02-12"></date-picker>

Range Dates

Enables a range of dates select

<date-picker :range="true"></date-picker>

Time Picker

Enables datetime picker

<date-picker :time="true"></date-picker>

Min & Max

Set the minimum date that a user can start picking from

<date-picker min="2017-01-01" max="2017-02-01"></date-picker>

Multiple Dates

Set the maximum date that a user can pick to

<date-picker :multiple="true"></date-picker>

Custom options

You can use all Flatpickr options by passing it to options property

<date-picker :options="opts"></date-picker>

<script>
    new Vue({
        el: '#app',
        data: {
            opts: {
                enableTime: true,
                enableSeconds: true
            }
        }
    })
</script>

Options

Property Description Type Default
value Initial selected date string/Date null
min The minimum date that a user can start picking from String/Date null
max The maximum date that a user can pick to String/Date null
time Enables time picker boolean false
range Enables a range of dates select boolean false
multiple Enables multiple dates select boolean false
options The Flatpickr config Options object {}
