Alertify

A cool Vue version replacement for javascript's alert 

<script>
    new Vue({
        el: '#app',
        methods: {
            alert () {
                this.$alertify({
                    title: 'Are you sure?',
                    text: 'Lorem ipsum dolor sit amet.',
                    type: 'warning',
                    showConfirm: true,
                    showCancel: true,
                    confirmText: 'Delete',
                    cancelText: 'Cancel'
                })
            }
        }
    })
</script>

Options

Property Description Type Default
title The title of alert string —
text the content of alert string —
showConfirm Whether to show confirm button boolean true
showCancel Whether to show cancel button boolean false
confirmText the confirm button text. string Confirm
cancelText the cancel button text. string Cancel
onConfirm the callback function when user click confirm button. If you return false, it will stop closing the window. function —
onCancel the callback function when user click cancel button. If you return false, it will stop closing the window. function —
